# path-finding-visualizer

Path Finding Visualizer (DSA Project)

This project is a Path Finding Visualizer developed as part of my Data Structures & Algorithms (DSA) course.
It visually demonstrates how classic graph traversal algorithms explore nodes and find a path between a start and an end point on a grid.

The application is built using Flask (Python) on the backend and uses a grid-based system where users can interactively place start/end nodes, add walls, and observe algorithm behavior step by step.
